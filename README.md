# Simple JS Snake Game
A lightweight, classic Snake game built entirely with HTML5 Canvas, CSS3, and Vanilla JavaScript. This project is contained within a single file for easy portability and learning.

# How to Run
No installation or local server is required.

Download/Copy the snake.html file.

Double-click the file to open it in any modern web browser (Chrome, Firefox, Safari, Edge).

# How to Play
Move: Use the Arrow Keys (Up, Down, Left, Right) to control the snake.

Objective: Eat the red food to grow your snake and increase your score.

Game Over: The game ends if the snake hits the border of the canvas or runs into its own body.

Restart: After a Game Over, an alert will show your score, and the page will automatically refresh to start a new game.

# Customization
If you want to modify the game, look for these variables in the <script> tag:

Speed: Change let gameSpeed = 100; (lower is faster).

Colors: Search for ctx.fillStyle to change the snake (green) or food (red) colors.

Size: Change const box = 20; to make the grid larger or smaller.
